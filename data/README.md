# Data

## Source

Download the dataset from Kaggle:

**[NBA Odds Data by Christopher Treasure](https://www.kaggle.com/datasets/christophertreasure/nba-odds-data?resource=download)**

## Setup

Place the downloaded CSV file in this directory:

```
data/
└── nba_odds_data.csv
```

## Contents

The dataset contains historical NBA game odds across multiple seasons, including:

- Home and away team identifiers
- Point spreads (home and away perspective)
- Game totals (over/under)
- Moneyline odds for home and away teams
- Sportsbook implied win probabilities
- Game outcomes (home win/loss)
